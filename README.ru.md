# ComfyUI Telegram Auto Send

**Language / Язык:** [🇺🇸 English](README.md) | 🇷🇺 Русский

---

Автоматически отправляет результаты всех нод **SaveImage** в Telegram-канал или чат — без изменений в рабочем процессе.

## Возможности

- **Не требует изменений в воркфлоу** — работает с любым существующим графом
- Включается/выключается прямо из панели **настроек ComfyUI**
- Отправка как **фото** (с сжатием Telegram) или как **файл** (оригинальный PNG без пережатия)
- **Бесшумная отправка** — доставка без звука и вибрации
- **Асинхронная очередь** — отправка никогда не блокирует следующую генерацию
- Подпись поддерживает плейсхолдеры `{time}` и `{date}`

## Установка

Клонируйте репозиторий в папку custom nodes ComfyUI и перезапустите ComfyUI:

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/GENKAIx/comfyui-telegram-auto-send
```

## Настройка

1. Откройте **Settings** (иконка шестерёнки ⚙️) в ComfyUI
2. Прокрутите до раздела **🤖 Telegram AutoSend**
3. Заполните поля:

| Настройка | Описание |
|---|---|
| **Enable auto-send** | Главный переключатель вкл/выкл |
| **Bot token** | Токен вашего бота от [@BotFather](https://t.me/BotFather) |
| **Chat ID / Channel ID** | ID чата или канала (например `-1001234567890`) |
| **Image caption** | Шаблон подписи — поддерживает `{time}` и `{date}` |
| **Send as file** | Отправить оригинальный PNG без пережатия Telegram |
| **Silent send** | Доставка без звука уведомления |

## Как это работает

1. Слушает событие `executed` ComfyUI после выполнения каждой ноды
2. Фильтрует вывод по `type = "output"` — **только SaveImage**, PreviewImage игнорируется
3. Читает сохранённый файл из папки вывода ComfyUI
4. Помещает задачу отправки в **постоянную фоновую очередь** — один поток-воркер обрабатывает задачи по порядку

## Требования

- ComfyUI (любая актуальная версия)
- Python-пакет `requests` (входит в комплект ComfyUI)
